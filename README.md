# تطبيق السيارات المتكامل (Car Hub Mobile Application)

تطبيق للهواتف المحمولة متعدد المنصات (Android & iOS) تم تطويره باستخدام إطار العمل **Flutter**، يمثل منصة متكاملة وذكية لبيع وشراء مقارنة السيارات، طلب خدمات الشحن، والتقديم على التسهيلات التمويلية (Financing / BNPL)، بالإضافة إلى توفير لوحة تحكم متطورة للوكلاء (Agents) والمشرفين (Admins).

---

## 🌟 الميزات الرئيسية للتطبيق (Key Features)

### 1. للمستخدمين والعملاء (Customers Area)
* **معرض السيارات الذكي:** تصفح كامل للسيارات المتوفرة مع استعراض شامل للمواصفات والأسعار العالية الدقة.
* **مقارنة السيارات (Car Comparison):** ميزة فريدة لمقارنة سيارتين أو أكثر برمجياً من حيث قوة المحرك، الأداء، استهلاك الوقود، والتقنيات المتاحة.
* **حلول التمويل والدفع بالتقسيط (Financing & BNPL):** 
  - إمكانية طلب التمويل المالي ورفع المستندات الثبوتية بشكل آمن (الهوية، كشف الحساب، تعريف بالراتب).
  - دعم بوابات الدفع الشهيرة مثل **Moyasar**، وخدمات التقسيط الرائدة **Tamara** و **Tabby**.
* **خدمات الشحن والتوصيل (Shipping Services):** إمكانية طلب شحن وتوصيل السيارات بين المدن والمناطق بسهولة.
* **نظام الإشعارات الفوري:** إرسال إشعارات وتحديثات لحظية حول حالة الطلبات أو العروض الجديدة.
* **المفضلة والمستندات:** حفظ السيارات المفضلة ورفع وتخزين المستندات بصورة آمنة.

### 2. للوكلاء وممثلي المبيعات (Agents & Representatives Space)
* **إدارة المخزون (Agent Inventory):** متابعة السيارات المتاحة وإضافتها أو تعديلها.
* **جدولة المواعيد والزيارات (Appointments):** تخطيط ومتابعة مواعيد العملاء لتجربة قيادة أو استشارات شراء.
* **بناء عروض الأسعار (Quote Builder):** إنشاء عروض أسعار رسمية ومخصصة وإرسالها للعملاء مباشرة.
* **تفاصيل ومعلومات العملاء:** استعراض متقدم لملفات العملاء وتاريخ اهتماماتهم لتسهيل المبيعات.

### 3. للمشرفين والإدارة العامة (Admin Control Panel)
* **إدارة المستخدمين والمناديب:** شاشات كاملة للتحكم في حسابات العملاء والوكلاء وممثلي المبيعات.
* **الإحصائيات والتحليلات (Dashboard Analytics):** رسوم بيانية ومخططات متقدمة توضح أداء المبيعات وحجم الأسطول عبر حزمة **fl_chart**.
* **إدارة أسطول السيارات:** إضافة وتحديث بيانات السيارات بشكل فوري.

## 🛠️ البنية البرمجية والتقنيات (Architecture & Tech Stack)

* **هيكل المجلدات (Architecture):** يعتمد المشروع على تصميم **Clean Architecture** مع تقسيم الميزات برمجياً (Feature-First Structure) لضمان سهولة التوسعة والصيانة.
* **إدارة الحالة (State Management):** استخدام حزمة **flutter_bloc** (Cubit) بشكل متكامل وفعال.
* **تحديد المواقع والمستندات (Permissions):** تم دمج نظام تحقق وإذن مسبق مخصص للمستندات والصور (الكاميرا والمعرض) والموقع بالاعتماد على `permission_handler` لتأمين تجربة مستخدم خالية من المشاكل.
* **التدويل ولغات التطبيق (Localization):** دعم كامل للغتين العربية والإنجليزية بواسطة **easy_localization**.
* **قاعدة البيانات المحلية:** استخدام **Hive** لتخزين البيانات المحلية وإعدادات التطبيق.
* **الاتصال بالشبكة:** استخدام حزمة **Dio** مع معالجات الأخطاء المتقدمة و نظام تسجيل البيانات (Logger).
* **الحماية والأمان:** يحتوي التطبيق على نظام كشف كسر الحماية للهواتف (Jailbreak/Root Detection) لحماية المعاملات المالية الحساسة.

## 🚀 تشغيل المشروع وتثبيته (Setup & Getting Started)

### المتطلبات الأساسية
- Flutter SDK (نسخة 3.8.1 أو أعلى).
- Android Studio / Xcode.

### خطوات التشغيل
1. قم بتحميل المستودع (Clone the repository).
2. قم بتنزيل الحزم البرمجية:
   ```bash
   flutter pub get
   ```
3. قم بتشغيل التطبيق:
   ```bash
   flutter run
   ```
